# AI文章生成助手

## 项目结构
```
src/
├── config/          # 配置文件
├── core/            # 核心功能模块
├── logs/            # 日志文件
├── models/          # AI模型文件
├── output/          # 生成结果输出
├── templates/       # 文章模板
├── ui/              # 用户界面
└── utils/           # 工具类
```

## 项目简介
AI文章生成助手是一个基于Python开发的单机软件，能够从PDF、图片等文件中提取文字，并利用AI技术生成格式化文章。

## 主要功能
- 支持PDF、PNG、JPG等多种格式文件的文字提取
- 中英文双语OCR识别
- 智能文章生成与排版
- 多种文章模板支持（学术论文、新闻稿等）
- 友好的图形用户界面

## 系统要求
- 操作系统：Windows 10/11 64位
- 内存：最低4GB，推荐8GB
- CPU：支持AVX指令集
- 硬盘空间：至少1GB可用空间

## 安装说明
1. 下载并安装最新版本的软件包
2. 运行安装程序，按提示完成安装
3. 首次运行时会自动下载必要的AI模型

## 使用方法
1. 启动程序
2. 点击"选择文件"按钮
3. 选择需要处理的文件
4. 选择输出文章模板
5. 点击"生成"按钮
6. 等待处理完成

## 注意事项
- 单个文件大小限制为20MB
- 仅支持常见文件格式（PDF、PNG、JPG）
- 请确保文件内容清晰可辨
